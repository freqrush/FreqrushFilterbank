FreqrushFilterbank by freqrush

A Sherman Filterbank 2 VST version, currently under development using "SynthEdit":http://www.synthedit.com , hopefully to be ported to Linux and Mac in the future.

After KVR's developer challenge 2007, I wanted to do another SynthEdit project, and found the Sherman Filterbank 2 manual at home between a pile of books. Somewhere in that manual, it's stated that this filter bank is too difficult to program digitally. So I got started to see if we would prove he's right, or that he's wrong. Anyway, I think a digital version of Sherman's Filterbank 2 would be great for people to learn how the "instrument" works.

Install

Extract the zip file into your VstPlugins folder, specified by your VstHost.
